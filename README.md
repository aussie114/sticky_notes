## Dependencies  
SDL3   

## About  
A program for making sticky notes on your desktop  

## Usage  
The program is driven by command line arguments with the first argument being the colour and proceeding  
arguemnts being each line of the note. ie ./sticky_notes "#FFFF88" "line1" "line2" "line3"  
You can pass in 0 as the first arguments for the default yellow.  
There is a configuration file for some more customization.
